# Global Climate Classification Change Analysis

Download, transform, explore, classify and visualize ERA5 climate data to compare a Recent Climate Period Average (2011-2020) with the WMO Climate Standard Reference Period Normals (1961-1990)
Introduction

This repository contains a multi-stage pipeline:

 + Download Data – Download ERA5 climate datasets.
 + Transform Data – Preprocess and derive monthly/seasonal stats.
 + Explore Data – Summarize, tabulate, and visualize trends.
 + Classify Data –
   +  Köppen Climate Classification (KCC)
   +  Trewartha Climate Classification (TCC)

Tools Used: R programming language, Positron IDE, Quarto publishing system
